## 🛠️ 3. 기본 호출 파라미터 (Common Schema)
공공데이터포털 공통 연동 규격에 따라 에이전트 연동 시 아래 파라미터 세트를 탑재합니다.

*   `serviceKey` (필수): 사용자 인증키 (`[한국관광공사 OpenAPI 인증키 - 마스킹 처리됨]`)
*   `MobileOS` (필수): `WEB` 또는 `ETC` (ennoia 플랫폼 구동 규격에 준함)
*   `MobileApp` (필수): `ennoia`
*   `_type` (옵션): `json` (LLM의 구조화된 응답 처리를 위해 JSON 호출 기본 설정 권장)
*   `numOfRows` (옵션): 한 페이지에 노출할 결과 레코드 수 (기본값: 10 ~ 20)
*   `pageNo` (옵션): 페이지 번호 (기본값: 1)
