프로젝트 비전 & 브랜드 스토리

여행을 뜻하는 영어 Tour는 원래 '돌다'라는 의미를 가집니다. 그러나 휠체어 이용자, 유모차 동반인, 그리고 거동이 불편한 고령층 실버 세대에게 세상은 단 몇 cm의 턱과 단 5도의 오르막길이라는 장벽으로 가로막힌 돌 수 없는 단절의 공간입니다.

크리넥스(Kleenex)는 보행 약자들이 여행지 현장에서 물리적 차단막을 만나 겪는 절망의 눈물을 부드럽게 닦아주고, 세상에서 가장 쾌적하고 뽀송뽀송한 배리어프리(Barrier-Free) 여행 경험을 선물하고자 태어난 LangGraph 기반 지능형 오케스트레이터입니다.

멀티 에이전트 직렬 파이프라인 (LangGraph Workflow)

단일 챗봇의 한계와 정보 환각(Hallucination)을 완전히 격파하기 위해, 역할을 극도로 분할하고 공유 대화 이력(Context sharing)을 극대화한 일자형 직렬 파이프라인을 구현했습니다.

1. Start (시작 노드)
전역 상태 변수(user_needs, target_region, weather_status)를 선언하여 흐름을 활성화합니다.
2. 크리넥스_기상_보안관 (기상 등급 판정) 기상청 API 연동
기상청 API 커넥터를 호출하여 실시간 자외선/체감온도 데이터를 입수한 뒤, 지식베이스와 대조해 등급 코드(NORMAL/CAUTION/EMERGENCY)를 weather_status에 갱신합니다.
3. 크리넥스_기획관 (무장애 코스 빌딩) 한국관광공사 MCP 연동
관광공사 MCP API를 연쇄 가동하여 단차 2cm 미만, 경사도 1:12 미만이 검증된 종로구 2박 3일 1차 큐레이션 일정 초안을 제작합니다.
4. 크리넥스_비상_우회관 (비상 실내 스왑) Backtracking 로직
앞선 대화 이력의 weather_status 코드를 감지합니다. CAUTION/EMERGENCY일 경우 야외길(청계천 등)을 즉각 감지해 100% 냉방과 엘리베이터가 보장된 실내관(MMCA, 역사박물관 등)으로 자동 대체합니다.
5. User approval (사용자 최종 승인)
물리 단차 및 실시간 기상 안전성이 교차 검증된 정합성 리포트를 사용자 대화창에 승인(Approve) 팝업과 함께 노출합니다.

에이전트별 상세 사양서 (Agent Specification)

각 에이전트의 뇌세포에 해당하는 시스템 프롬프트 명세입니다. 오동작을 줄이기 위해 온도를 0.3(저온 설정)으로 엄격하게 제한했습니다.

# 역할 (Persona)
너는 기상청 실시간 API를 활용해 보행 약자의 야외 이동 안전도를 평가하는 '크리넥스_기상_보안관'이다.

# 동작 가이드라인 (Rules & Constraints)
1. 사용자가 질문하면 기상청_자외선지수_조회 도구를 가동해 실시간 자외선 지수(UV Index) 데이터를 확보하라.
2. 지식베이스의 임계치(WE_001~WE_006)를 대조하여 안전 등급을 판정하라.
   - 자외선 0 ~ 5 (NORMAL)
   - 자외선 6 ~ 7 (CAUTION)
   - 자외선 8 이상 (EMERGENCY)
   - 초미세먼지 75 ㎍/㎥ 이상 (EMERGENCY)
   - 체감온도 32°C 이상 (EMERGENCY)
3. 판정된 코드와 날씨 브리핑만 [출력 형식]에 맞게 출력하라.
# 역할 (Persona)
너는 보행 약자들을 위해 100% 안전하고 쾌적한 무장벽 여행 동선 초안을 기획하는 '크리넥스_기획관'이다.

# 동작 가이드라인 (Rules & Constraints)
1. 연동된 크리넥스_무장애_지식베이스의 보행약자 세그먼트별 물리 장벽(ID: BF_001~BF_008)을 최우선 기준으로 삼아 정합성을 평가하라.
2. 관광공사 MCP 도구를 5회 이하로 제한하여 호출(detailWithTour2 등은 유력 후보 3~4곳만 실행)함으로써 LangGraph 재귀 제한 25회를 초과하지 않게 조율하라.
3. 획득한 관광지의 단차, 엘리베이터 여부를 대조해 안전한 무장벽 코스 1차 초안을 작성하라.
# 역할 (Persona)
너는 기상 보안관의 위험 판정을 파악해, 기획관이 짠 일정을 100% 안전한 냉방 실내 전시 코스로 조율하는 전술 전문가 '크리넥스_비상_우회관'이다.

# 동작 가이드라인 (Rules & Constraints)
1. 기상 등급 판정이 "CAUTION" 또는 "EMERGENCY"인 경우에만 이 우회 프로토콜을 가동한다. (NORMAL은 무수정 통과)
2. 기획관이 짠 일정 중 야외 노출 동선을 선별하여 100% 무단차 및 냉방이 완비된 'MMCA 국립현대미술관', '대한민국역사박물관', '서울공예박물관' 등의 명품 실내 코스로 자동 대체(Swap)하라.
3. 사용자에게 최종 출력 시, 우회된 사유와 변경된 최종 뽀송 일정을 친절히 렌더링하라.

크리넥스 무장애 RAG 지식베이스 규격

안전 임계치 검증의 신뢰성을 담보하기 위해 구축하여 연동시킨 knto_barrier_free_kb.csv의 명세 정보입니다. 총 14개 클래스에 달하는 촘촘한 배리어프리 대응 데이터셋을 탑재하고 있습니다.

ID (식별자) 카테고리 대상 보행약자 물리 장벽 / 기상 임계치 조건 상세 기준 및 우회 요령
BF_001 수동휠체어_단독 수동 휠체어 이용자 (보조인 없음) 문턱 1.5cm 이하, 보행로 경사도 4.7도 이하 아스팔트, 우레탄 선호. 경복궁 박석/자갈길 기피. 주출입구 무단차 및 배리어프리 화장실 필수 동선 배치.
BF_003 전동휠체어 전동 휠체어 및 전동 스쿠터 이용자 문턱 2.0cm 이하, 보행로 경사도 8.1도 이하 진흙길, 모래사장 진입 차단. 배터리 소진에 대비한 전동 휠체어 급속 충전기 위치(예: 안녕인사동) 매핑 필수.
BF_006 시각장애인 시각장애인 및 저시력 관광객 점자 블록 및 음성 유도기 안내 필수 계단 진입 전 점자 블록 확인. 오디오 가이드 대여소 및 해설사 투어 가능 구역(경복궁, 창덕궁 등) 최우선 배정.
BF_008 임산부_배려 임산부 및 태아 안전 관리 필요군 연속 보행 30분 미만, 리프트 필수 급작스러운 오르막길 회피 및 엘리베이터 동선 위주 매핑. 수유실 및 기저귀 교환대 등 모성 보호 휴게 공간 연계.
WE_002 자외선_위험 보행 약자 및 고령층 전체 실시간 자외선 지수 8 이상 낮 시간대 야외 활동 전면 금지. 청계천 도보길과 경복궁 마당 관람을 삭제하고 MMCA 서울 등 100% 실내 관람 대체.
WE_003 체감온도_폭염 고령층 실버 세대 및 열사병 고위험군 체감온도 32도 이상 야외 연속 보행 15분 미만 통제. 동선 중간에 냉방 대형 복합몰(디타워, 그랑서울 등) 휴식 최소 1시간 강제 배치.
WE_004 미세먼지_위험 호흡기 질환자 및 보행 노약자 초미세먼지(PM2.5) 75 ㎍/㎥ 이상 야외 전통 명소 일정을 전면 취소하고, 고성능 공기 정화 환기 설비를 완비한 대형 실내 미술관·박물관 코스로 자동 치환.

핵심 기술적 시행착오 & 극복기 (Trials & Triumphs)

🚀 LangGraph 재귀 제한 25회 돌파 (Recursion Limit)

현상: 무장애 검증 프롬프트가 가동되었을 때, AI가 API 목록 중 수십 개의 관광지를 발견하여 모든 후보지에 대해 detailWithTour2 API 조회를 재귀 루프로 돌려 랭그래프 엔진의 한계선인 25단계(Recursion limit of 25)를 넘어서 작동을 강제 중단시키는 에러가 빈번했습니다.

해결: 캔버스를 건드리지 않고 에이전트의 프롬프트 내에 'API 5회 이하 예산 Throttling 규칙'을 이식하여, AI가 첫 관광지 목록에서 무장애 시설이 잘 갖춰져 있을 것으로 추정되는 핵심 3~4곳만 지능적으로 골라내고 이들만 상세 검증하게 만듦으로써 속도 500% 개선과 더불어 완벽한 안정성을 실현했습니다.

🔌 API 커넥터 화이트스페이스 미세 튜닝

현상: 기상청 생활기상지수 API를 등록할 때 복사-붙여넣기 문자열 마지막에 들어간 보이지 않는 미세 개행문자(\n)나 공백 때문에 엔노이아의 정적 URL 밸리데이터가 "올바른 URL 형식이 아닙니다"를 출력하며 등록을 거부했습니다.

해결: URL 입력창의 문자열 우측 끝 문자 }에 커서를 완벽하게 밀착하는 백스페이스 백트래킹 방식을 적용해 공백을 완전히 박멸하여 커넥터 저장 및 실시간 통신에 최종 성공했습니다.
