## ⚙️ 3. 캔버스 노드별 최종 세팅 상태 (Node Configuration Details)

### 1) Start 노드 (전역 변수 설정)
*   **상태 변수**:
    *   `state.user_needs` = `"휠체어 이용 보행 약자"`
    *   `state.target_region` = `"서울 종로구"`
    *   `state.weather_status` = `"NORMAL"`

### 2) 크리넥스_기상_보안관 노드 (Weather Sheriff Node)
*   **역할**: 실시간 기상청 자외선 정보 수집 및 보행 약자 기상 안전 평가.
*   **시스템 프롬프트**:
    > 너는 기상청 실시간 API를 활용해 보행 약자의 야외 이동 안전도를 매의 눈으로 감시하고 평가하는 '크리넥스_기상_보안관'이다. 자외선 지수 0~5는 "NORMAL", 6~7은 "CAUTION", 8 이상은 "EMERGENCY" 코드를 출력 형식에 맞춰 반환하라.
*   **RAG 지식베이스**: `크리넥스_무장애_지식베이스` (혼합 검색 / Top-K: `6`)
*   **API 커넥터**: `기상청_자외선 지수_조회` 연동 (자동 실행)
*   **파라미터**: `dataType=JSON`, `areaNo=1111000000` (기본값), `time=2026060213` (기본값)
*   **온도**: **`0.3 (정확함)`**
*   **상태 저장**: 에이전트의 출력을 `state.weather_status` 변수에 매핑.

### 3) 크리넥스_기획관 노드 (Curation Planner Node)
*   **역할**: 물리 단차 및 경사로 임계치를 검증하여 1차 동선 초안 빌딩.
*   **시스템 프롬프트**:
    > 너는 보행 약자들을 위해 100% 안전한 무장벽 여행 동선을 큐레이팅하는 플래너 '크리넥스'이다. RAG 지식베이스의 물리 장벽 허용 한계(BF_001~BF_005)와 기상 임계치(WE_001~WE_003)를 준수하고, KTO MCP 도구를 활용해 무장애 상세 정보가 적합한 핵심 후보지만 엄선하여 일정표 초안을 마크다운 표 형태로 생성하라.
*   **RAG 지식베이스**: `크리넥스_무장애_지식베이스` (혼합 검색 / Top-K: `6`)
*   **MCP 도구**: `한국관광공사_공용_MCP` (전체 63개 도구 / 자동 실행)
*   **온도**: **`0.3 (정확함)`**

### 4) 크리넥스_비상_우회관 노드 (Rerouter Node)
*   **역할**: 기상 보안관 판정 코드(`CAUTION`/`EMERGENCY`) 감지 시 야외 코스를 냉방 완비 실내 무단차 박물관/미술관 코스로 스왑.
*   **시스템 프롬프트**:
    > 너는 기상 보안관의 경보를 감시하여 기획관의 야외 일정을 100% 무단차 실내 전시 코스로 대체하는 우회관이다. CAUTION/EMERGENCY일 경우 야외 동선을 찾아 MMCA, 대한민국역사박물관, 서울공예박물관 실내관으로 대체하고 사유를 밝혀라. NORMAL인 경우 우회 없이 일정을 통과시켜라.
*   **RAG 지식베이스**: `크리넥스_무장애_지식베이스` (혼합 검색 / Top-K: `6`)
*   **MCP 도구**: `한국관광공사_공용_MCP` (전체 63개 도구 / 자동 실행)
*   **온도**: **`0.3 (정확함)`**

### 5) User approval 노드 (최종 정합성 검증 확인 게이트)
*   **역할**: 사용자에게 안전 필터링이 100% 완료되었음을 공지하고 승인 버튼 노출.
*   **메시지 전문**:
    > 제안해 드린 일정 후보지들에 대해 한국관광공사 무장애 인프라 데이터(물리 단차 2.0cm 이하 검증) 및 기상청 실시간 기상 안전 지수(자외선 및 체감온도 평가)를 100% 교차 정합성 검증 완료했습니다.  
    >   
    > 아래 **[승인(Approve)]** 버튼을 클릭하시면, 크리넥스가 최종 보증하는 뽀송뽀송하고 안전한 2박 3일 맞춤형 무장벽 일정을 화면에 출력해 드립니다.
