Pain Point & Market Needs

세상에서 가장 외롭고 불안한 여행자, 보행 약자

♿물리적 장벽 (Physical Barrier)

거동 불편 보행 약자는 전체 인구의 약 25%에 육박합니다. 정보 미비로 현장 단차와 가파른 경사로 앞에서 깊은 절망을 마주합니다.

"경복궁에 도착하고 나서야 자갈길과 턱 때문에 휠체어 단독 진입이 불가능함을 깨달았습니다."

☀️기상 이변 장벽 (Weather Barrier)

기존 큐레이팅 AI는 보행 약자가 기상 이변(자외선, 폭염)에 훨씬 더 취약하다는 점을 무시하며, 야외 장시간 노출은 신체적 위험을 유발합니다.

"체감온도 32도가 넘는 폭염 속 청계천 도보길을 걷다가 부모님이 탈진하셨습니다."

💡 해결책: 공공데이터와 지능형 에이전트 오케스트레이션을 결합해 물리/기상 장벽을 100% 사전에 차단하는 AI 어시스턴트

System Canvas Architecture

LangGraph 기반의 실시간 3단계 오케스트레이션 엔진

단일 챗봇의 한계와 환각을 완전히 제어하기 위해, **역할이 격리된 3개의 전문 에이전트 파이프라인**을 직렬 연결하여 견고한 오케스트레이션을 구현했습니다.

⚙️ 3대 에이전트 & 노드 역할

  • Start 노드: 전역 변수 및 사용자 여행 요구사항 수립.
  • 기상 보안관 (Weather Node): 기상청 API 실시간 등급 판정.
  • 크리넥스 기획관 (Planner Node): 관광공사 무장애 API 연동 1차 설계.
  • 비상 우회관 (Rerouter Node): 기상 위험 시 실내 코스로 자동 대체.
  • User Approval (승인 노드): 정합성 최종 승인 및 뽀송 일정 출력.
⚡ 핵심 메커니즘: 기상 등급이 EMERGENCY로 판단되는 즉시, 비상 우회관 노드가 개입하여 야외 동선을 냉방 및 휠체어 리프트가 완비된 실내 박물관·미술관으로 자동 스왑(Swap) 조치합니다.

Trials & Triumphs

더 견고하고 안정적인 시스템을 향한 엔지니어링 집념

시행착오 A

LangGraph 재귀 제한 초과 에러

❌ 현상: 에이전트가 50개 이상의 관광지 정보를 가져와 하나하나 API 루프를 돌며 배리어프리 상태를 검증하다가, LangGraph의 노드 재귀 제한 한도인 25회(Recursion limit)를 넘어 시스템이 중단되는 에러 발생.

✅ 극복: 프롬프트에 '도구 호출 예산 제약(Throttling)' 규칙을 수립하여 1차 연동 필터링을 통해 무장애 신뢰도가 가장 확실한 핵심 후보지 3~4곳만 엄격히 엄선하여 루프를 돌도록 설계. 에러 원천 차단 및 응답속도 대폭 개선.

시행착오 B

API 커넥터 미세 공백 URL 파싱 에러

❌ 현상: 엔노이아 플랫폼 내의 API URL 설정 인풋창에 미세한 공백문자나 눈에 보이지 않는 줄바꿈 기호(\n)가 복사-붙여넣기 과정에서 흘러들어가 API 도구 활성화 자체가 실패하고 500 Internel Error가 발생하는 현상.

✅ 극복: 플랫폼 내부 커넥터가 아닌 백엔드 실행 노드의 호출 로직 상에서 문자열 뒤쪽 보이지 않는 제어 문자를 제거(Trimming)하고, 수동으로 바인딩 정합성을 검증하는 에러-레질리언스(Resilience) 가드레일 레이어를 프롬프트와 오케스트레이션에 수동 패치함.

NORMAL Scenario

경관과 역사 문화를 만끽하는 평탄한 뽀송 일정

⚙️ 동적 오케스트레이션 동작 흐름

  • 기상청 실시간 API 판정: 자외선 보통(NORMAL) 및 체감온도 양호 등급 확인.
  • 배리어프리 기획관 동작: 평탄 보행로(무단차, 경사로 완비) 중심의 야외 전통 명소 일정 설계.
  • 비상 우회관 즉시 패스: 기상 이변이 없으므로 일정을 변경하지 않고 유지.
  • 최종 뽀송 일정 도출 및 승인 대기: - 경복궁 광화문 경사로 진입 및 근정전 박석 우회 안내
    - 인사동 쌈지길 전층 경사로 순환식 관람 동선 매핑
    - **[최종 정합성 검증]** 사용자의 직접 승인(Approve)을 획득하기 위해 대화창 내 인터랙티브 팝업 대기 노드 활성화

EMERGENCY Scenario

기상 경보 시 100% 안전한 실내 동선 자동 치환

⚙️ 위험 기상 시 우회 치환 로직

  • 기상청 API 실시간 위험 감지: 자외선 '매우 높음(8 이상)' 또는 체감온도 32°C 초과 시 즉각 EMERGENCY 등급 판단.
  • 비상 우회관의 즉각 개입: 기획관의 야외 코스(청계천 도보 책, 조계사 경내 등)를 감지하고 강제 제어권 획득.
  • 실내 명소 강제 스왑(Swap): 100% 냉방이 보장되고 휠체어 램프/리프트가 구비된 실내 미술관·박물관 코스로 변환.
  • 최종 뽀송 대피 일정 도출: - MMCA 서울 실내 현대미술 특별전 관람
    - 대한민국역사박물관 실내 엘리베이터 동선
    - 서울공예박물관 공예 전시 및 휴게시설 연계
