*   **If/else 로직 분기 노드**: 조건식 `state.weather_status != "NORMAL"` 판정 및 동적 라우팅
*   **크리넥스_비상_우회관**: `비상우회관_4` (LLM: Claude 3.5 Sonnet / 온도 `0.3` / 위험 기상 시 야외 코스를 무단차 실내 전시 코스로 스왑)
*   **User approval**: 사용자 승인 인터랙티브 게이트 (Approve ➡️ 대화 종료, Reject ➡️ Set state 리셋)
*   **Set state**: 거절 세션 초기화 (`state.weather_status` ➡️ `"NORMAL"`, `state.target_region` ➡️ `""`)
*   **크리넥스_안내관**: `안내관_1` (LLM: Claude 3.5 Sonnet / 온도 `0.3` / 시스템 리셋 안내 배너 출력 후 안전 종료)

* **연동 예제 스크립트 (Python)**:
```python
import requests
import json

def request_chat_completion():
    url = "https://api.ennoia.so/api/llm-orchestrator/chat/stream/4630528296/5"
    
    headers = {
        "project": "KNTO-PROMPTON-2026-391",

        "Content-Type": "application/json; charset=utf-8"
    }
    
    # 예시: 보행 약자 세그먼트 요건 및 목적지 입력 payload (messages 배열 및 variables 전송)
```
