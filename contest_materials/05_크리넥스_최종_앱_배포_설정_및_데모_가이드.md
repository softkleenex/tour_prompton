## 🧭 2. 심사위원을 위한 핵심 데모 시나리오 (QA Test Scenarios)

공개 공유 링크 접속 후, 대화창에서 아래 3가지 유형의 질문을 던져 실시간 기상/물리 장벽 정합성 검증 엔진의 동작을 직접 확인할 수 있습니다.

### 🌟 시나리오 A: 보행 약자 유형에 따른 정밀 큐레이팅
*   **추천 프롬프트**: `"서울 종로구 2박 3일 휠체어 이용 코스 짜줘"`
*   **관전 포인트**:
    1.  RAG 지식베이스(`BF_001` 수동 휠체어 단독)의 문턱 1.5cm, 경사 4.7도 기준 인용.
    2.  경복궁 박석(돌길) 노면 구간 우회 가이드 안내.
    3.  인사동/청계천 코스 내 전동 휠체어 급속 충전기 위치와 장애인 화장실 정보 자동 검증 출력.

### ☀️ 시나리오 B: 폭염/자외선 위험 상황의 실내 코스 자동 우회 (Rerouting)
*   **추천 프롬프트**: `"고령의 부모님 모시고 가는데 오늘 자외선 지수가 위험 수준이래. 동선 어떻게 짜야 해?"`
*   **관전 포인트**:
    1.  기상청 실시간 자외선지수 API를 동적으로 호출하여 자외선 등급 판정.
    2.  RAG 지식베이스의 폭염 기준(`WE_002` 자외선 8 이상) 작동 확인.
    3.  경복궁 야외 관람, 청계천 산책로 등의 동선을 **100% 무단차 실내 냉방 전시(국립고궁박물관, 서울공예박물관 등)**로 우회 교체 및 사유 설명.
    4.  **`User approval`** 노드가 발동되어 화면에 **[Approve(승인)] / [Reject(거절)]** 버튼 팝업이 깔끔하게 로출되는 인터랙티브 연출 확인.

## 🐍 6. 외부 백엔드 API 연동 명세 및 예제 코드 (PPT 발표자료 첨부용)

본 서비스는 향후 카카오톡 챗봇, 모바일 앱 등의 외부 채널(Front-end)로 즉각적인 이관 및 상용화가 가능하도록 백엔드 API 파이프라인으로 완전 릴리즈되었습니다. (※ 소개서 PPT 후반부의 '확장 비전 및 상용화 방안' 슬라이드에 이 명세를 포함하십시오.)

* **API 엔드포인트 URL (stream)**:
  `https://api.ennoia.so/api/llm-orchestrator/chat/stream/4630528296/5`
* **HTTP Method**: `POST`
* **연동 예제 스크립트 (Python)**:
```python
import requests
import json

def request_chat_completion():
    url = "https://api.ennoia.so/api/llm-orchestrator/chat/stream/4630528296/5"
    
    headers = {
        "project": "KNTO-PROMPTON-2026-391",
        "apiKey": "[엔노이아 OpenAPI 인증키 - 마스킹 처리됨]",
        "X-ENNOIA-USER-ID": "test_user_id",  # MCP 서버 구동을 위해 유저 고유 ID 헤더 지정 필수
        "Content-Type": "application/json; charset=utf-8"
    }
    
    # 예시: 보행 약자 세그먼트 요건 및 목적지 입력 payload (messages 배열 및 variables 전송)
    payload = {
        "messages": [
            {
                "role": "user",
                "content": "서울 종로구 2박 3일 휠체어 이용 코스 짜줘"
            }
        ],
        "variables": {
            "user_needs": "휠체어 이용 보행 약자",
            "target_region": "서울 종로구",
            "weather_status": "NORMAL"
        }
    }
    
    response = requests.post(url, headers=headers, json=payload, stream=True)
    
    print(f"Status Code: {response.status_code}")
    for chunk in response.iter_lines():
        if chunk:
            print(chunk.decode('utf-8'))

if __name__ == "__main__":
    request_chat_completion()
```
