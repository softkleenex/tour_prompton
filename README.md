## 🌐 3. 핵심 연동 OpenAPI 리스트 (허가 완료)

크리넥스 1인 팀은 이미 공공데이터포털을 통해 **총 28개의 한국관광공사 OpenAPI의 활용 신청 및 승인을 100% 완료**했습니다. 그중 크리넥스 에이전트의 중추가 될 핵심 API 5선은 다음과 같습니다.

1.  **국문 관광정보 서비스_GW (ID: 15101971)**: 전국 관광지, 맛집, 숙박 시설 원천 정보 제공.
2.  **무장애 여행 정보 (ID: 15144030)**: 장애인 주차장, 휠체어 대여, 엘리베이터 설치 등 보행 약자 맞춤 편의시설 정보 제공.
3.  **관광지 집중률 방문자 추이 예측 정보 (ID: 15128555)**: 향후 30일간의 머신러닝 기반 관광지 혼잡성 상대 수치 예측 정보 제공.
4.  **관광지별 연관 관광지 정보 (ID: 15128560)**: TMAP 내비게이션 기반 관광지별 연계 방문 음식점 및 숙소 이동도 데이터 제공.
5.  **관광사진 정보 서비스_GW (ID: 15151868)**: 에이전트 대화창에 고해상도 경관 시각화 이미지를 동적 렌더링하기 위한 데이터 제공.
